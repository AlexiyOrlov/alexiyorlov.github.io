This article instructs how to install JavaFX and run applications requiring it.

Linux

  • Download SDK for linux from Gluon.
  • Create an environment variable for Javafx runtime using terminal: export PATH_TO_FX=[absolute path to its folder]/lib
  • Test the runtime by running an application using Javafx: java -jar --module-path $PATH_TO_FX --add-modules javafx.controls some.jar
