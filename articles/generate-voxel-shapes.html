How to generate Minecraft block voxel shapes

Did you know that you don't have to calculate the shape of the block manually? There is a plugin for Blockbench called Voxelshape Generators that can generate a block shape from its model. Here is how to do it.

  1. Open the plugin menu (File -> Plugins)
  2. Search for Voxelshape generators and install it
  3. Create a group in your model named 'VoxelShapes'
  4. Place all model parts that you want to generate a shape from into this group
  5. Then select File -> Export -> Export voxel shape
  6. It will create a text file with a Java function that generates a shape for your block
  7. Use this function in your code.
