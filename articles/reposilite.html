Setting up Reposilite maven repository

In this article I will show how you can set up Reposilite on your computer.

  1. Download Maven application from Official website in the form of archive.
  2. Extract arhive contents into an empty directory
  3. Add path of the 'bin' directory to the system PATH variable.
  4. Test maven by 'mvn -v'. It should show some info
  5. Create a new directory and open it
  6. Execute 'mvn archetype:generate' to create a blank project
  7. download Reposilite, place it in empty directory and run from terminal
