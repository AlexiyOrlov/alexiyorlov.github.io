This is a documentation for "Generators". "Generators" is a technological mod, aiming to simplify, alleviate and speed up various activities in the game. Most of the blocks and items it adds require energy to operate, but there are exceptions. Currently Generators use energy API "Energies", which is open-source and ready for use by other mods.

Generators adds a tool enchantment "Multi-break", which works like vein-mining.

    Mods integration:
  • Some recipes of the machines can be viewed in REI, as well as fuel values for generators.
  • If you have ModMenu, you'll be able to edit the config from it.
