This block auto-crafts a specified item from incoming ingredients, with high speed.

The crafting pattern is on the top of the GUI.

You can set the recipe by clicking slots with appropriate items. Output slot will auto-update if the pattern matches some recipe. In the picture the crafter is set to craft iron blocks. To clear the pattern, click on the slots with empty cursor.

Crafter has a single inventory where it takes ingredients from and outputs result in. It is sided so the process can be automated.
