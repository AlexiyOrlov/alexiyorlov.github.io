Shredder produces 1 sugar and 2 Bagasse from 1 sugar cane. Bagasse can be used as a fuel or processed into paper using Paper Mill.

Shredder uses minecraft's system of recipes - by reading them from Json files. If you want to create additional recipes, you must do it the datapack way.

  1. Create a folder for the datapack.
  2. Create file inside the folder named "pack.mcmeta". Example: 
    
            {
              "pack" : {
                    "pack_format" : 4,
                    "description" : "Test shredder recipes"
                }
            }
    "Description" can be any sensible info about the recipes.
  3. Create folders "data/[mod/pack identifier]/recipes".
  4. Navigate into "recipes" folder.
  5. Create Json recipe files (format will be described below).
  6. Zip the "data" and "pack.mcmeta" into one archive.
  7. Copy the zip file (now a data pack) into directory "minecraft/saves/[World name]/datapacks".
  8. Launch the game and check whether your recipes were added.

Recipe format

Take a look at original shredder recipe: 

 
        {
          "type": "generators:shredder",
          "input": "minecraft:sugar_cane",
          "outputs": [
            {
              "item": "generators:bagasse",
              "count": 2
            },
            {
              "item": "minecraft:sugar"
            }
          ]
        }
The "type" value must be "generators:shredder"; "input" value is the item identifier; "outputs" is an array of 1 or 2 result itemstacks; "count" is the size of the stack and is optional.
